Ponto de verificação 3.6

Use a notação para listas e use os seus itens corretamente.

Use corretamente a estrutura e os itens das listas. Evitar o uso destes elementos para formatar parágrafos. Quando necessário use listas numeradas.

Exemplo correto

1. Lista de instrumentos encontrados numa banda de rock:

  1. Guitarra
  2. Baixo
  3. Bateria

E o código para este exemplo é:

<ol>

<li>Lista de instrumentos encontrados numa banda de rock:</li>

<ol>

<li>Guitarra</li> <li>Baixo</li> <li>Bateria</li>

  </ol>

</ol>

Exemplo errado

• Lista instrumentos encontrados numa lista de banda de rock:

Guitarra

Baixo

Bateria

 

E o código para este exemplo é:

<ul>

<li>Lista de instrumentos encontrados numa banda de rock: <dl>

<dt>Guitarra <dt>Baixo <dt>Bateria </dl>

</ul>
