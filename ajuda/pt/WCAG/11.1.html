Ponto de verificação 11.1

Utilizar tecnologias do W3C sempre disponíveis e adequadas a uma determinada tarefa; utilizar as versões mais recentes, desde que suportadas.

O principal motivo para manter suas páginas dentro dos padrões do W3C é que com isto sua página estará sempre dentro dos padrões de acessibilidade propostos pelo WAI.

Apesar de muitos comandos do HTML 4.0 ainda não estarem amplamente suportados por todos os navegadores, não é problema para deixar de usar. Há uma grande preocupação em fazer com que estes navegadores estejam cada vez mais dentro dos padrões do W3C, embora estas implementações ocorram de forma gradativa.
