Ponto de verificação 13.2

Fornecer metadados para acrescentar informações semânticas a páginas ou sítios.

Disponibilize metadados para facilitar mais ainda a navegação e a procura de informações dentro de suas páginas. Elementos: META, ADDRESS, TITLE, LINK.

<head>

<meta name=”keywords” content=”acessibilidade, usabilidade, webdesign” />

<meta name=”description” content=”Curso de Acessibilidade na web” />

</head>

<address> Rua Conde de Lages, 44 - sala 502 - Glória </address>

Por exemplo: você pode usar o comando LINK REL para criar relações entre os documentos. Esse tipo de associação é interessante para usuários de LINX (é um navegador só de texto).

<head>

<link rel=”anterior” href=”chk9-0.htm” />

<link rel=”índice” href=”overchk.htm” />

<link rel=”próximo” href=”chk11-0.htm” />

</head>
