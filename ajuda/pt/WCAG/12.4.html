Ponto de verificação 12.4

Associar explicitamente os rótulos aos respectivos controles.

Use o comando LABEL para associar campos nos formulários. Fazendo isso os leitores de tela associarão os elementos dos formulários de forma correta.

Veja o exemplo abaixo:

 

Ele pode parecer confuso, mas usando o comando LABEL, como no código abaixo, isso fará com que as pessoas que usem leitores de tela não tenham problemas ao ler o formulário.

<fieldset>

<legend align=”top”>Options Required</legend><br />

<input type=”checkbox” name=”opt1” ID=”opt1” value=”abs” /><br />

<label for=”opt1”>Anti-lock breaks</label><br />

<label for=”opt2”>Moon roof </label>

<input type=”checkbox” name=”opt2” ID=”opt2” value=”mr” /><br />

<input type=”checkbox” name=”opt3” ID=”opt3” value=”hm” />

<label for=”opt3”>Heated mirrors</label>

<label for=”opt4”>Power windows</label><br />

<input type=”checkbox” name=”opt4” ID=”opt4” value=”pw” />

</fieldset>
