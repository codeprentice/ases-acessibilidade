Ponto de verificação 3.7

Marcar as citações. Não utilizar marcações de citação para efeitos de formatação, como, por exemplo, o avanço de texto.

Elemento <Q>:

Use o elemento Q para marcar citações. O elemento Q ainda não é suportado por alguns navegadores (Internet Explorer por exemplo). Neste caso coloque também a indicação de citação manualmente.

“Vou para casa à meia-noite!”

<q lang=”pt”>"Vou para casa à meia-noite!"</q>

 

 

Elemento <BLOCKQUOTE>:

O elemento BLOCKQUOTE também é usado para citações. Mas o que acontece na maioria das vezes por ele fazer um recuo do texto ele acaba sendo usado como forma de tabular parágrafos. Deixe de usar o BLOCKQUOTE como forma de tabular textos. Para isso, use folha de estilos.

Exemplo

No capítulo 7, Bob deixa as suas palavras mais importantes:

Vou para casa à meia-noite!

No capítulo 7, Bob deixa as suas palavras mais importantes: <blockquote> Vou para casa à meia-noite!</blockquote>

O exemplo abaixo é feito em CSS e faz o mesmo efeito que o comando BLOCKQUOTE.

<p style=”margin-left: 10%; margin-right: 10%; >Bob foi para casa...</p>

Bob foi para casa à meia-noite, porque o trem noturno tem uma freqüência bem inferior ao verificado no horário diurno. Ele estava preocupado para não adormecer mas pelo menos queria arranjar um lugar sentado.

Ou, numa declaração de estilo de seção,

P.inset {margin-left: 10%; margin-right: 10%;}

e depois no HTML:

<p class=”inset”>Bob foi para casa...</p>
