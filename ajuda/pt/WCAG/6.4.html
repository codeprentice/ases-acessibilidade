Ponto de verificação 6.4

Em programas interpretáveis e applets, assegurar que a resposta a eventos seja independente do dispositivo de entrada.

Sempre que tiver script associe o elemento NOSCRIPT.

Aqui está o código usado para criar um botão “scripted”:

<a href=”javascript: pop()”>Ganhe um prêmio!</a>

E o código da função script propriamente:

<script type=”text/javascript”> function pop() { alert (“Cadastre-se agora!”) } </script>

<noscript>Ganhe um Premio! Siga para a página de cadastro.</noscript>

Obs: o atributo language=”JavaScript” ficou deprecated
