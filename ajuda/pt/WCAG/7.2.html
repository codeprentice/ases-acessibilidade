Ponto de verificação 7.2

Evitar situações que possam provocar o piscar do conteúdo das páginas (isto é, alterar a apresentação a intervalos regulares, como ligar e desligar), possibilitar que o software do usuário possa ter o controle desse efeito.

Se um conteúdo piscar (por exemplo: um cabeçalho que aparece e desaparece em intervalos regulares) é utilizado, forneça um mecanismo para parar o efeito de piscar.

Se sente que tem que dar destaque a algum texto causando que o mesmo flash (ligue e desligue), então use o CSS, atributo “text-decoration: blink” . Ao usar CSS vai ser possível ao usuário parar o efeito, desativando a folha de estilo, ou eliminar a regra numa folha de estilo personalizada.

Não use os elementos BLINK e MARQUEE. Estes elementos não fazem parte de qualquer especificação W3C para HTML (i.e., eles não são elementos normalizados).
