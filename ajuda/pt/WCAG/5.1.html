Ponto de verificação 5.1

Em tabelas de dados, identificar os cabeçalhos de linha e de coluna.

<table border=1>

<caption>Exemplo de uma tabela de dados simples criada com notação HTML.</caption>

<thead>

<tr>

<td></td>

<th>Cabeçalho Col. 1</th>

<th>Cabeçalho Col. 2</th> </tr>
</thead>
<tbody>
<tr>

<th>Cabeçalho linha 1</th>

<td>C1L1</td>

<td>C1L2</td> </tr>
<tr>

<th>Cabeçalho linha 2</th>

<td>C2L1</td>

<td>C2L2</td> </tr>
</tbody>
</table>

 

Nota: no caso de utilizar tabelas para formatar páginas (em vez de CSS), então NÃO deve usar a notação existente para tabelas de dados (como TH, HEADER, SCOPE, COLGROUP, etc.) e isto porque aqueles elementos devem ser usados por alguns agentes para identificar e manipular os dados.
