Ponto de verificação 9.1

Fornecer mapas de imagem armazenados no cliente ao invés de no servidor, exceto quando as regiões não puderem ser definidas por forma geométrica disponível.

Outro método de criar uma mapa de imagens “client side” é através da utilização dos elementos <OBJECT>, <MAP> e <A> (embora este método não seja ainda largamente suportado):

Este código deve apresentar links padrão caso o navegador do usuário não suporte o OBJECT ou o carregamento de imagens se encontre desligado:

<object data=”imgmap4.gif” type=”image/gif” usemap=”#map1”>

<map name=”map1”>

<p>Navegue no sítio. </p>
