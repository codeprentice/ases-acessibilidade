Ponto de verificação 11.3

Fornecer informações que possibilitem aos usuários receber os documentos de acordo com as suas preferências (por ex., por idioma ou por tipo de conteúdo).

Em vez de incluir links tais como “Aqui está a versão francesa deste documento”, use negociação de conteúdos de forma a que a versão Francesa seja apresentada de acordo com o requerido pelos clientes desta versão dos documentos.

  1. Em HTML 4.0 muitos elementos permitem especificar o idioma do conteúdo com o atributo ‘lang=”.
  2. Se não for possível usar a negociação de conteúdos, use o ‘hreflang’ em HTML com os elementos <A> ou <LINK> para identificar o idioma dos documentos alvo.

<a href="versao.htm" hreflang="FR">
