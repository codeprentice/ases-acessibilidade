Ponto de verificação 3.4

Utilizar unidades relativas, e não absolutas, nos valores dos atributos da linguagem de marcação e nos valores das propriedades das folhas de estilo.

Em CSS não use valores absolutos como “pt” ou “cm” e sim valores relativos como o “em”, "ex" ou em porcentagem.

Exemplo

Ao invés de usar:

.Texto { font-size: 14pt; }

passe a usar:

.Texto { font-size: 1em; }
