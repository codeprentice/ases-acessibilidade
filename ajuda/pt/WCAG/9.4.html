Ponto de verificação 9.4

Criar uma seqüência lógica de tabulação para percorrer links, controles de formulários e objetos.

A tecla TAB é usada também, dentro do navegador para se movimentar dentro das páginas. Com isso elas devem ser dispostas para que sejam navegadas de forma linear. O comando TABINDEX permite que você estabeleça uma ordem lógica. O TABINDEX trabalha com os elementos <A>, <AREA>, <BUTTON>, <INPUT>, <OBJECT>, <SELECT> e <TEXTAREA>.

Utilização de TABINDEX:

A distribuição/disposição dos campos no formulário do exemplo que se segue está intencionalmente embaralhada. No entanto, o código de TABINDEX coloca uma ordem lógica para os campos; ao usar a tecla TAB eles surgem na ordem “correta” (para navegadores que suportam o atributo tabkey - de fato, este exemplo específico apenas funciona (ou trabalha tal como o autor espera) se estiver usando o Internet Explorer 4.0 ou posterior):

<form method=”post” action=”home.htm”>

      <input type=”submit” value=”Enviar” tabindex=”6” /><br /> Qual é a sua cor favorita?<br />

Vermelho <input type=”checkbox” name=”CB1” value=”CheckBox1” tabindex=”3” /><br /> Verde <input type=”checkbox” name=”CB2” value=”CheckBox2” tabindex=”4” /><br />
Azul <input type=”checkbox” name=”CB3” VALUE=”CheckBox3” tabindex=”5” /><br />
Nome da empresa <input type=”text” name=”Tboxin” tabindex=”2”  size=”20” /><br />

      <input type=”reset” value=”Apagar” tabindex=”7”><br />

<a href=”form-alt.htm” tabindex=”1”>Salte  este formulário para ver as

      alternativas acessíveis</a>

</form>

Apesar de poder controlar a ordem da tecla TAB, o ideal é que você crie os formulários de forma lógica.
