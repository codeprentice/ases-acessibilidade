Ferramenta Associadora
de Descrição para Imagens

Associa descrições de imagens (obrigatórias às normas de acessibilidade) àquelas imagens desprovidas das tais encontradas durante a análise.

Decrição da Ferramenta :

  • Ferramenta responsável pela identificação e inserção (feita pelo usuário) da falta de descrição em imagens.

URL: Endereço da página que se deseja avaliar.

Código em Fonte:

  • Código em Edição:
    • É exibido o código fonte editável, como um editor de HTML.
  • Código Original:
    • Exibe o código original, apenas para comparação, pois este não é editável.

Regra: Regra que obriga a todas as imagens possuirem um correspondente alternativo em texto para os cegos.

Digite a Descrição a para a imagem: O usuário deve digitar aqui a o conteúdo alternativo para a imagem.

Aplicar: Aplica o conteúdo alternativo inserido na imagem.

Lista de Erros: Lista todos os erros de sintaxe encontrados no código. E, ao clicar sobre o erro, o usuário é direcionado à sua ocorrência no código editável.

Salvar: Salva o arquivo alterado.

Abrir: Abre, e avalia automaticamente, o arquivo HTML.

Tela Anterior: Retorna à tela anterior a este avaliador.

Reverter: Recarrega o código original, sem edição alguma.
