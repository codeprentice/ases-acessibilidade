Recomendação 1.20 - Assegurar que todas as páginas possam ser utilizadas mesmo que os  programas interpretáveis, os applets ou outros objetos programados tenham sido desativados  ou não sejam suportados. Sempre que tiver script associe logo a seguir o elemento “noscript”.

Aqui está o código usado para criar um botão “scripted”:

  1. <a href=”javascript: pop()”>Cadastre-se agora!</a>   

 

E o código da função script propriamente:  

  1. <script language=”JavaScript” type=”text/javascript”>   
  2. function pop() { alert (“Você Ganhou o Primeiro Prémio!”) }
  3. </script>    
  4. <noscript> 
  5. Você Ganhou o Primeiro Prémio! Siga para o cadastro do sítio.  </noscript>
