Recomendação 2.7 - Sempre que existir uma linguagem de marcação apropriada, utilizar  marcações em vez de imagens para transmitir informações, um exemplo é a linguagem  MathML que permite a criação de fórmulas matemáticas somente utilizando-se das tags  apropriadas.
Exemplo: No caso de equações matemáticas use MathML para sua construção e folhas de  estilo para formatar o texto e controlar o layout. Evite também usar imagens para representar  texto.

  9. <mi>x</mi>
  10. </mfrac></mrow>
  11. <annotation-xml encoding=”MathML-Content”>
  12. <apply><int/> 
  13. <bvar><ci>x</ci></bvar>
