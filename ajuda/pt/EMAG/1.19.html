Recomendação 1.19 - Assegure a acessibilidade de objetos programados, tais como  programas interpretáveis e applets, garantindo que a resposta a eventos seja independente do dispositivo de entrada e que qualquer elemento dotado de interface própria possa funcionar  com qualquer leitor de tela ou navegador que o usuário utilize. Evite colocar scripts que estejam vinculados a links, se isso não for possível, fornecer informações equivalentes em uma  página alternativa acessível.

NÃO use: <A href=”javascript: pop()”>Cadastre-se agora!</A>

Evite a criação de links que usem “javascript” tais como URL. Se um usuário não usar scripts,  então não será capaz de encontrar os links, uma vez que o navegador não consegue criar o conteúdo link. Isto é um link considerado como “beco sem saída” para um navegador em que os scripts não são suportados ou gravados.
