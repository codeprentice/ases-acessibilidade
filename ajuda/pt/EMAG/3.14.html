Recomendação 3.14 - Forneça metadados para acrescentar informações semânticas e descritivas do site, que sejam úteis para os mecanismos de busca.

Pode-se usar o comando “link rel” para criar relações entre os documentos. Esse tipo de  associação é interessante para usuários de LINX (navegador texto). 

  1. <head>
  2.   <link rel="anterior" href="chk9-0.htm" />
  3.   <link rel="proximo" href="chk11-0.htm" />
  4.   <link rel="indice" href="overchk.htm" />
  5. </head>
