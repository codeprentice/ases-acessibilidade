Recomendação 2.17 – Forneça informações sobre como o site está estruturado, através de um mapa ou de sumário. Crie o mapa de forma textual, associando aos títulos das páginas para não causar confusão aos usuários que utilizarem leitores de tela.

O mapa ou sumário do site pode mostrar a hierarquia das páginas, isso é feito utilizando o atributo “title” dentro do comando “a”.

No código html:

  1. <a href="a.htm" title="nível 1">Página de boas-vindas</a>
  2. <a href="b.htm" title="nível 2">Página de entrada</a> (página principal) <a href=”c1.htm” title=”nível 3”>Produtos</a>
  3. <a href="c.htm" title="nível 4">Sistemas</a>
  4. <a href="d.htm" title="nível 5">Sistema 1000</a>
