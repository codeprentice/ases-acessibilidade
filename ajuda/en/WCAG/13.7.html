Checkpoint 13.7

If search functions are provided, enable different types of searches for different skill levels and preferences.

Keyword Search

In this kind of search the user types a word that the search mechanism serachs in the whole site, by pages that has the typed word.

Complex Search

In this search, the user uses some boolean operators (and, or). In case the search doesn't find anything, type similar words.
You can also dispose of search engines like Yahoo or Google have.
