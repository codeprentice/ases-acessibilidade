Checkpoint 6.4

scripts and applets, ensure that event handlers are input device-independent.

Always that you have a script associate it to a NOSCRIPT element.

Here is some code used to create a “scripted” buttom:

<a href=”javascript: pop()”>Win a prize!</a>

And the fucntion code:

<script type=”text/javascript”> function pop() { alert (“Register now!”) } </script>

<noscript>Win a prize! Go to the registration page.</noscript>

Obs: the language=”JavaScript” attribute is deprecated
