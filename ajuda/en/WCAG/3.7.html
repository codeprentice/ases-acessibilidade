Checkpoint 3.7

Mark up quotations. Do not use quotation markup for formatting effects such as indentation.

<Q> element:

Use the Q element to mark up de citations. The Q element is not supported by a few browsers (Internet Explorer for example). In this case, also put the indication of citation manually.

“At midnight, I'll get home!”

<q lang=”pt”>"At midnight, I'll get home!</q>

 

 

<BLOCKQUOTE> element :

The BLOCKQUOTE element is also used to citations. But what happens most of the times is to be used to tabulate paragraphs. Do not use the BLOCKQUOTE element to tabulate texts. To do that, use CSS code.

Example:

This example marks up a longer quotation with BLOCKQUOTE:

   <BLOCKQUOTE cite="http://www.example.com/loveslabourlost">
     <P>Remuneration! O! that's the Latin word for three farthings.
        --- William Shakespeare (Love's Labor Lost).
     </P>
   </BLOCKQUOTE>
