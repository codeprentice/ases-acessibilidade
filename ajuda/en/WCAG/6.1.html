Checkpoint 6.1

Organize documents so they may be read without style sheets. For example, when an HTML document is rendered without associated style sheets, it must still be possible to read the document.

Example:

Imagination is more

important

than intelligence

Albert Einstein

This example uses the style attributes to specity the vertical and horizontal positions:

<style type=”text/css”> <!--

.parte1 /* Imagination is more */

{ padding-left: 0; margin-top: 10px; color: red; font-size: 18pt; font-family: times

new roman }

.parte2 /* important */

{padding-left: 100px; margin-top: 40px;color: black; font-size: 14pt; font-family:

times new roman }

.parte3 /* than intelligence */

{ padding-left: 200px; margin-top: 70px; color: blue; font-size: 20pt; font-family:

times new roman }

.parte4 /* Albert Einstein */

{ padding-left: 350px; margin-top: 100px; color: green; font-size: 16pt; font-family:

times new roman }

--> </style>

You can arrange the text on the screen by letting the commands to format them, you will have a well structured HTML content, as followed:

<div class=”parte1”>Imagination is more</div>
<div class=”parte2”>important</div>
<div class=”parte3”>than intelligence</div>
<div class=”parte4”>Albert Einstein</div>

And so, if the CSS code is disabled or they are not supported by your browser, it will read:

Imagination is more important than intelligence Albert Einstein
