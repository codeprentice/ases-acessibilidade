Checkpoint 9.3

For scripts, specify logical event handlers rather than device-dependent event handlers.

  1. Use “onmousedown” with “onkeydown”.
  2. Use “onmouseup” with “onkeyup”
  3. Use “onclick” with “onkeypress”
  4. Note that there is no keyboard equivalent to double-click (“ondblclick”) in HTML 4.0.

Example:

In this case, ONFOCUS is perfectly apropriated, because it works with mouse or keyboard:

<button onfocus=”pop1()”> Register! </button>

And here is the script:

<script type=”text/javascript”>

function pop1() { alert (“Compete to 10 prizes!”) }

</script>
