Checkpoint 3.6

 

Mark up lists and list items properly.

Use correctly the items list structure. Avoid this elements use to format paragraphs. When necessery use numbered lists.

Correto example:

1. List of musical instruments found in a rock band:

  1. Guitar
  2. Bass
  3. Drum

And the correct code example is:

<ol>

<li>List of musical instruments found in a rock band:</li>

<ol>

<li>Guitar</li> <li>Bass</li> <li>Drum</li>

  </ol>

</ol>

Wrong example:

• List of musical instruments found in a rock band:

Guitar

Bass

Drum

 

And the wrong code example is:

<ul>

<li>List of musical instruments found in a rock band: <dl>

<dt>Guitar <dt>Bass <dt>Drum </dl>

</ul>
