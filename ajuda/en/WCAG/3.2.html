Checkpoint 3.2

Create documents that validate to published formal grammars.

In case is it necessary to develop a page with a particular grammar, use the DOCTYPE instruction at the first line code of your HTML file. This kind of instruction tells the servers, browsers and validators that the code is inside the informedlanguage rules. In the example the code is according to the HTML 4.0 Transitional rules. The DTD corresponds the grammar, "Document Type Description".
 

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
  <!DOCTYPE HTML PUBLIC “-//W3C//DTD HTML 4.0 Transitional//EN”>
