Checkpoint 5.1

For data tables, identify row and column headers.

 

<table border=1>

<caption>Data table example made with HTML notation.</caption>

<thead>

<tr>

<td></td>

<th>Header Col. 1</th>

<th>Header Col. 2</th> </tr>
</thead>
<tbody>
<tr>

<th>Header row 1</th>

<td>C1L1</td>

<td>C1L2</td> </tr>
<tr>

<th>Header row 2</th>

<td>C2H1</td>

<td>C2H2</td> </tr>
</tbody>
</table>

 

Note: in case you use table to format pages (instead of CSS), do NOT use the data table notation (as TH, HEADER, SCOPE, COLGROUP, etc.) because those elements must be used by a few agents to identify and manipulate the data.
