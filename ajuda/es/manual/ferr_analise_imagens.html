Ferramenta Análise Sistemática
para Imagens

Ferramenta que descobre a ocorrência da falta de descrição de uma mesma imagem em diversas páginas distintas.

Descrição da Ferramenta:

  • Esta ferramenta percebe a repetição de uma mesma imagem sem o conteúdo alternativo e lista todas as ocorrências para que o usuário possa alterá-las automaticamente.

Regra: Regra que obriga a todas as imagens possuirem um correspondente alternativo em texto para os cegos.

Digite a Descrição para uma Imagem: Neste campo o usuário deve digitar a descrição da imagem. Ex.: Fotografia de uma torta.

Lista de Erros: Exibe todos os erros encontrados.

Salvar Modificados: Salva todos os conteúdos alternativos que foram modificados.

Salvar Última Página: Salva apenas a última página alterada.

Aplicar a Página: Aplica o conteúdo alternativo que foi digitado em todas as suas ocorrências da página.

Aplicar a Todos: Aplica a todos as ocorrências de erros (todas as páginas).

Painel Anterior: Retorna ao painel anterior.

Detalhamento sobre a Funcionalidade Salvar Modificados:

Ao clicar em Salvar Modificados será apresentada uma janela que permite escolhermos um diretório onde será criada uma estrutura de pasta similar a encontrada no site analisado. para cada página salva são criados dois arquivos, possuem o mesmo nome que o da pagina salva diferindo apenas na extensão, sendo um deles em txt, que contem o caminho correto do local original da página salva, permitindo que possamos recolocar o arquivo corrigido no local correto, e o de extensão html, que contem o código corrigido.

Por motivo de algumas paginas virem com caracteres não permitidos para nomes de arquivos, alguns caracteres destes nomes precisam ser convertidos para o número correspondente da tabela ASCII, estes estarão entre vírgulas no nome do arquivo

exemplo:

arquivo que contem o endereço original da pagina:

End_47,.html.txt

arquivo que contem o código reparado:

,47,.html

note que o 47 está entre virgulas, isso significa que se trata de um caractere ASCII

na tabela abaixo se descobre que se trata do caractere '/'

Observação: Para utilizar essa ferramenta, o usuário deve primeiro avaliar o site (ou página) pela página principal do ASES, o Avaliar de Acessibilidade, e então ir em: Ferramentas > Conteúdo Alternativo para TAGS > IMG, após ter o relatório da avaliação. Somente assim será possível utilizar o recurso de análise sistemática de imagens.
