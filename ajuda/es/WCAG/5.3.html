Punto de verificación 5.3

No utilice tablas para maquetar, a menos que la tabla tenga sentido cuando se alinee. Por otro lado, si la tabla no tiene sentido, proporcione una alternativa equivalente (la cual debe ser una versión alineada ). [Prioridad 2] Nota. Una vez que las aplicaciones de usuario soporten la colocación mediante hojas de estilo, las tablas no se deben utilizar para maquetar.
